# Forensics, GRC & Advanced Crypto

In this module, we'll explore the more advanced concepts of forensics, GRC, and advanced cryptography.

## Digital Forensics
- **Evidence Collection Basics:** Collecting and preserving digital evidence (e.g., using write blockers, imaging tools).
- **Disk & Memory Forensics (Intro):** Analyzing disk and memory images to identify security issues.

## Governance & Compliance
- **ISO 27001 Basics:** An international standard for information security management systems (ISMS).
- **Risk Management:** A systematic process for identifying and managing security risks.
- **Security Policies:** Designing and implementing secure policies (e.g., using access control, encryption, incident response).

## Advanced Cryptography
- **Digital Signatures:** A mathematical scheme for verifying the authenticity and integrity of digital messages or documents.
- **Key Exchange Mechanisms:** Protocols for securely exchanging encryption keys (e.g., Diffie-Hellman).
- **Certificate Management:** Managing digital certificates (e.g., using PKI, SSL/TLS).

## Practical Tip
When performing digital forensics, always follow a methodology (e.g., NIST SP 800-86) to ensure that you're covering all the necessary security controls.
