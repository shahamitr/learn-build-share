# Security Concepts & Tools

In this module, we'll explore the core concepts and tools used in cybersecurity.

## Basic Security Concepts
- **Authentication vs Authorization:** Authentication is the process of verifying who you are (e.g., username/password), while authorization is the process of verifying what you have access to.
- **Encryption Basics:** Symmetric encryption uses the same key for encryption and decryption, while asymmetric uses a public key for encryption and a private key for decryption.
- **Password Security & Hashing:** Strong passwords are long, complex, and unique. Hashing is a one-way process that converts a password into a fixed-length string of characters.

## Tools (Intro Level)
- **Command Prompt / Terminal:** The command-line interface for interacting with your operating system.
- **Wireshark (Just Overview):** A network protocol analyzer that allows you to capture and interactively browse the traffic running on a computer network.
- **Antivirus / Firewall Basics:** Antivirus software detects and removes malware, while a firewall monitors and controls incoming and outgoing network traffic.

## Practical Tip
When creating passwords, always use a password manager to store and generate strong, unique passwords for all your accounts.
