# 10. GitHub Profile & Contribution Activity

## Your Developer Portfolio

Your GitHub profile is your resume.

### Contribution Graph

The grid of green squares shows your activity over the last year.
*   **Commits**
*   **PRs**
*   **Issues**

### Profile README

You can create a special repository named exactly like your username (e.g., `username/username`). The `README.md` from this repo will appear on your public profile page.

## Lab: Create Profile README

1.  **Create Repo:** Create a new public repository named exactly the same as your GitHub username.
2.  **Initialize:** Check **Add a README file**.
3.  **Edit:** Add some content about yourself:
    *   👋 Hi, I’m @username
    *   👀 I’m interested in ...
    *   🌱 I’m currently learning ...
4.  **Commit:** Save the file.
5.  **Check Profile:** Go to your profile page (`github.com/username`). You should see your new README displayed prominently.
