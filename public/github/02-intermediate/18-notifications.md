# 18. GitHub Notifications and Watch Settings

## Notification Management

GitHub can be noisy. Manage your notifications to stay focused.

### Repository Watching

*   **Watch:** Receive notifications for all conversations.
*   **Participating and @mentions:** Only receive notifications when you are mentioned or participate.
*   **Ignore:** Never be notified.

### Custom Notifications

You can configure notifications for specific events (e.g., only releases, only security alerts).

## Lab: Configure Notification Preferences

1.  **Watch Button:** Locate the **Watch** button (top right of repo).
2.  **Select:** Choose **Custom**.
3.  **Events:** Select **Issues** and **Pull requests**.
4.  **Apply:** Click **Apply**.
5.  **Global Settings:** Go to your profile settings -> **Notifications** to manage email and web notifications globally.
