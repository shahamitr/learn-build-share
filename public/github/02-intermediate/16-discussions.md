# 16. GitHub Discussions

## Community Discussions

Discussions are a space for open-ended conversations, questions, and ideas that aren't necessarily actionable tasks (issues).

### Feature Proposals

Use Discussions to brainstorm new features before creating formal issues.

### Q&A

Users can ask questions, and community members can provide answers. You can mark an answer as the "Accepted Answer".

## Lab: Start a Discussion Thread

1.  **Enable:** Go to **Settings** -> **General** -> **Features** and check **Discussions**.
2.  **Discussions Tab:** Click the **Discussions** tab.
3.  **New Discussion:** Click **New discussion**.
4.  **Category:** Select **General**.
5.  **Topic:** "Ideas for next version".
6.  **Body:** "What features would you like to see?"
7.  **Start:** Click **Start discussion**.
