# Rust Syntax Basics

Rust is a powerful systems programming language that focuses on safety and performance. Understanding its syntax is essential for building scalable and maintainable applications.

## Key Concepts

- **Variables:** Declare variables using `let` and `const`.
- **Data Types:** Use primitive types like `i32`, `f64`, `bool`, and `char`.
- **Functions:** Define functions using `fn`.
- **Control Flow:** Use `if`, `else`, `match`, and `loop`.
