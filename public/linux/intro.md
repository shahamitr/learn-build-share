# Introduction to Linux

Linux is a family of open-source Unix-like operating systems based on the Linux kernel, an operating system kernel first released on September 17, 1991, by Linus Torvalds.

## Why Linux?
- **Open Source:** Free to use and modify.
- **Security:** Highly secure and less vulnerable to malware.
- **Stability:** Known for running for years without needing a reboot.
- **Customization:** Highly customizable with various desktop environments and distributions.

## Common Distributions
- Ubuntu
- Debian
- Fedora
- CentOS
- Arch Linux

In this course, you'll learn the basics of navigating and managing a Linux system using the command line.
