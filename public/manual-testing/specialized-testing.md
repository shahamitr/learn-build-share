# Mobile, Security & Performance Testing

In this module, we'll explore more specialized testing types.

## Advanced Testing Types
- **Performance Testing (Concept):** Testing how the system performs under load.
- **Security Testing (Basics):** Identifying vulnerabilities in the system.
- **Accessibility Testing:** Ensuring that the application is accessible to all users.
- **Compatibility Testing (Devices, OS):** Testing that the application works across different devices and operating systems.

## Mobile Testing
- **Android vs iOS Testing Basics:** Understanding the differences between the two platforms.
- **Emulator vs Real Device:** Testing on both emulators and real devices.
- **Mobile-specific Bugs:** Identifying bugs that are specific to mobile devices.

## Practical Tip
When testing on mobile, always verify that the application handles network interruptions gracefully.
