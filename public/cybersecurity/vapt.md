# Security Architecture & VAPT

In this module, we'll explore the more advanced concepts of security architecture and VAPT.

## Advanced Security Domains
- **Network Security Architecture:** Designing and implementing secure networks (e.g., using firewalls, IDS/IPS, VPNs).
- **Cloud Security (AWS / Azure Basics):** Securing cloud-based resources (e.g., using IAM, security groups, encryption).
- **Application Security (Secure Coding Principles):** Designing and implementing secure applications (e.g., using input validation, output encoding, secure authentication).

## Advanced Ethical Hacking
- **Vulnerability Assessment & Penetration Testing (VAPT):** A systematic process for identifying and exploiting vulnerabilities in a system.
- **Exploitation Techniques:** Using tools like Metasploit to exploit vulnerabilities in a system.
- **Privilege Escalation:** Gaining higher-level access to a system (e.g., from a user account to an administrator account).
- **Post-exploitation:** Performing actions on a system after it has been compromised (e.g., exfiltrating data, maintaining access).

## Practical Tip
When performing VAPT, always follow a methodology (e.g., PTES) to ensure that you're covering all the necessary security controls.
