# Amazon RDS Basics

Amazon Relational Database Service (Amazon RDS) makes it easy to set up, operate, and scale a relational database in the cloud. It provides cost-efficient and resizable capacity while automating time-consuming administration tasks such as hardware provisioning, database setup, patching, and backups.

## Supported Database Engines

*   Amazon Aurora
*   PostgreSQL
*   MySQL
*   MariaDB
*   Oracle Database
*   SQL Server

## Key Features

*   **Managed Service:** AWS manages the underlying infrastructure and database software.
*   **Multi-AZ Deployments:** Provides high availability and failover support for DB instances.
*   **Read Replicas:** Makes it easy to take advantage of elastic capacity beyond the capacity constraints of a single DB instance for read-heavy database workloads.
*   **Automated Backups:** Automated backups of your DB instance.
