# Antigravity CLI

The Antigravity CLI is a powerful tool designed to help you quickly scaffold and manage your antigravity projects. It provides a set of commands to generate boilerplate code, manage dependencies, and run simulations.

## Creating a New Project

To create a new Antigravity application, you can use the `create-antigravity-app` command via `npx`. This will set up a new project with the recommended directory structure and default configurations.

```bash
npx create-antigravity-app my-app
```

This command will:
1. Create a new directory named `my-app`.
2. Initialize a new Node.js project.
3. Install the core Antigravity dependencies.
4. Generate a basic project structure, including configuration files and starter code.

## Project Structure

A typical Antigravity project generated by the CLI looks like this:

```
my-app/
├── src/
│   ├── index.js        # Main entry point
│   ├── config.js       # Configuration settings
│   └── simulations/    # Directory for simulation scripts
├── package.json        # Project metadata and dependencies
└── README.md           # Project documentation
```

## Running Simulations

Once your project is set up, you can start writing and running simulations. The CLI provides commands to execute your simulation scripts within the Antigravity environment.
