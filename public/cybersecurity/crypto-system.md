# Cryptography & System Security

In this module, we'll explore the more advanced concepts of cryptography and system security.

## Cryptography (Intermediate)
- **Hashing Algorithms (MD5, SHA):** One-way functions that convert data into a fixed-length string of characters.
- **Public Key Infrastructure (PKI):** A system for managing digital certificates and public-key encryption.
- **SSL/TLS Basics:** Secure Sockets Layer (SSL) and Transport Layer Security (TLS) are protocols for establishing a secure, encrypted connection between a device and a network.

## System Security
- **Hardening Windows & Linux:** Configuring systems to be more secure (e.g., disabling unnecessary services, enabling firewalls).
- **User Roles & Permissions:** Managing access to system resources based on user roles and permissions.
- **Patch Management Basics:** Keeping systems up-to-date with the latest security patches.

## Basic Scripting
- **Bash or Python Basics:** Learning the basics of scripting for security tasks (e.g., automating network scans, parsing log files).

## Practical Tip
When hardening a system, always follow a security baseline (e.g., CIS Benchmarks) to ensure that you're covering all the necessary security controls.
