# Networking & CI/CD

In this module, we'll explore the more advanced features of Kubernetes networking and CI/CD.

## Advanced Networking
- **CNI (Container Network Interface):** A standard for configuring network interfaces for containers.
- **Service mesh (intro):** A way to manage communication between services in a cluster.
    - **Istio:** A popular service mesh for Kubernetes.

## CI/CD Integration
- **Deploying with pipelines:**
    - **GitLab CI/CD:** Using GitLab to deploy and manage your application.
    - **Jenkins:** Using Jenkins to deploy and manage your application.

## Practical Tip
When integrating with CI/CD, always use a version control system to manage your application's configuration and use a CI/CD pipeline to deploy it.
