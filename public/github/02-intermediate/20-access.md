# 20. Managing Repository Access

## Collaborators

You can invite individual users to collaborate on your repository.

### Teams

In an Organization, you can create teams (e.g., "Developers", "Designers") and grant access to repositories by team.

### Permission Levels

*   **Read:** Can clone and pull.
*   **Triage:** Can manage issues and PRs but not write code.
*   **Write:** Can push to repository.
*   **Maintain:** Can manage repository settings.
*   **Admin:** Full access.

## Lab: Add Collaborators

1.  **Settings:** Go to **Settings** -> **Collaborators**.
2.  **Add People:** Click **Add people**.
3.  **Search:** Search for a friend's username (or a second account of yours).
4.  **Invite:** Select them and click **Add to this repository**.
5.  **Pending Invite:** They will receive an email invitation to accept.
