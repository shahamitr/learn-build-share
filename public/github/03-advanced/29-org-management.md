# 29. Organization-Level Management

## Teams

Create teams to manage access for groups of people. Mention teams (`@org/team-name`) to notify everyone.

### Permissions

*   **Base permissions:** Default access for all members.
*   **Repository access:** Grant specific access to teams or individuals.

### Organization Settings

*   **Billing:** Manage payment.
*   **Member privileges:** Control what members can do (e.g., create repos).
*   **Verified domains:** Verify ownership of your domain.

## Lab: Setup Team Permissions (Requires Org)

1.  **Create Org:** (Optional, free) Create a new Organization.
2.  **Create Team:** Go to **Teams** -> **New team**. Name it "Developers".
3.  **Add Members:** Invite a friend.
4.  **Add Repo:** Go to a repository -> **Settings** -> **Collaborators and teams**.
5.  **Add Team:** Add "Developers" with **Write** access.
