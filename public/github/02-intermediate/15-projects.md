# 15. GitHub Projects (Project Management)

## Kanban Boards

GitHub Projects allows you to organize issues and PRs into a Kanban board (To Do, In Progress, Done).

### Task Tracking

You can convert issues into cards on the board.

### Milestones

Milestones group issues and PRs by a target date (e.g., "v1.0 Release").

## Lab: Create Project Board

1.  **Projects Tab:** Go to the **Projects** tab in your repository.
2.  **New Project:** Click **New project** -> **Board**.
3.  **Name:** "Development Roadmap".
4.  **Add Item:** Click **+ Add item**. Type "Implement login page" and press Enter.
5.  **Convert to Issue:** Click the `...` on the item and select **Convert to issue**.
6.  **Move:** Drag the card from **Todo** to **In Progress**.
